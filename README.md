# 감정일기 Emotion Diary

# 🌐 Intelligent Web Design

사용자가 하루의 감정을 이모티콘과 글, 이미지로 기록할 수 있는 개인 감정 기록 웹 애플리케이션입니다.<br>
JSP, MySQL, Apache Tomcat 기반으로 동작하며, 간단한 UI와 직관적인 감정 기록 인터페이스를 제공합니다.

---

## 📌 프로젝트 개요

감정은 하루 동안 수많은 변화가 생기지만, 대부분 기록되지 않은 채 흘러가곤 합니다.<br>
이 프로젝트는 사용자가 쉽고 자연스럽게 오늘의 감정과 생각을 시각적으로 기록할 수 있도록 돕는 것을 목표로 합니다.

사용자는 감정을 이모티콘으로 선택하고, 텍스트로 상세 내용을 작성하며, 이미지 업로드를 통해 자신의 감정을 더 생생하게 표현할 수 있습니다.<br>
기록된 감정 일기는 날짜와 함께 리스트로 조회할 수 있으며, 수정 및 삭제 기능도 제공합니다

---

## ✅ 주요 기능

- 🧑‍💻 **회원가입 / 로그인 기능**  
  사용자 식별 및 개인 감정 데이터 관리

- 🙂 **감정 선택 UI (이모티콘 5종)**  
  한눈에 이해되는 감정 선택 인터페이스 제공

- 📝 **감정 일기 작성**  
  감정 선택 → 텍스트 작성 → 이미지 업로드(선택)

- 🌗 **다크모드 전환**  
  밝은 테마 / 어두운 테마 선택 가능

- 🖼️ **이미지 업로드 기능**<br>
  Apache FileUpload로 구현된 파일 저장 기능 포함

- 📃 **일기 목록 페이지**<br>
  날짜/시간 순 정렬, 이미지 썸네일 미리보기 지원

- 🔍 **상세 페이지 / 수정 / 삭제**<br>
  작성한 감정 일기의 전체 내용을 확인하고 필요 시 수정 및 삭제 가능

- 🌐 **다국어 기능 (한국어 / 영어)**<br>
  상단 우측 언어 선택 기능 제공

- 🔑 **최근 로그인 ID 기억 기능**<br>
  UX를 고려한 편의 기능
